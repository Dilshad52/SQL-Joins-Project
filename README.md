# SQL-Joins-Project
# Customer & Order Analytics Dashboard

## Project Overview

This project demonstrates SQL Join concepts used in real-world business analytics.

## Skills Used

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- SELF JOIN
- SQL Analytics
- Power BI Dashboarding

## Business Questions Solved

- Which product generates the highest revenue?
- Who are the top customers?
- Which customers have not placed any orders?
- How is revenue distributed across products?

## Key Insights

- Laptop is the highest revenue-generating product.
- Revenue is concentrated among a few customers.
- Some customers remain inactive and need engagement strategies.

- SQL
